Paratize

Provides a set of functions that does parallel processing on collection of data or functions.

Summary↑

pc_each(arg_list, fun, kargs \\ [])

Provides a handy function to do parallel processing of each via chunks. It calls the fun with each argument. The pool_size default is size of arg_list. Setting pool_size to :schedulers will use the number of system cores
pc_exec(fun_list, kargs \\ [])

Provides a handy function to do parallel processing of functions via chunks. It calls each fun in fun_list and returns the list of result. Keyword list is supported. Ordering of result is assured via sequential Task.await on collected Task pids. The pool_size default is size of arg_list. Setting pool_size to :schedulers will use the number of system cores
pc_map(arg_list, fun, kargs \\ [])

Provides a handy function to do parallel processing of map via chunks. It calls the fun with each argument and returns the list of result. Ordering of result is assured via sequential Task.await on collected Task pids. The pool_size default is size of arg_list. Setting pool_size to :schedulers will use the number of system cores
